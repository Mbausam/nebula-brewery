SECTION 1 - GAME OVERVIEW
1. Game Overview
1.1 Concept Statement
Nebula Brewery is a space-themed idle clicker game where players brew stardust, build cosmic infrastructure, and explore the universe - one galaxy at a time. Rooted in the addictive DNA of Cookie Clicker, it layers in real astronomical education, absurdist humor, and an ever-expanding universe that never truly ends.
Design Philosophy
The game should feel like you accidentally learned astrophysics while trying to stop clicking. Every fact is real. Every joke is earned. Every upgrade makes you want the next one immediately.

1.3 Core Loop
The game loop is deceptively simple and deliberately infinite:
Click the nebula/star  earn Stardust (?)
Spend Stardust  buy Buildings that auto-generate Stardust per second
Spend Stardust  buy Upgrades that multiply production
Spend Stardust  buy Spacecraft that unlock Exploration Zones
Explore Zones  earn new ingredients, unlock real space facts, gain production bonuses
Watch the News Ticker  get rewarded with absurd cosmic headlines and hidden upgrades
Prestige (Big Bang Reset)  restart with permanent multipliers, new content unlocks
Repeat forever. The universe is endless. So is the game.
1.4 The Secret Educational Mission
Nebula Brewery has a hidden second purpose: it is a space education platform in disguise. Players will:
Encounter real astronomical facts dressed as game flavor text
Learn about actual spacecraft, celestial bodies, and cosmic phenomena through zone discovery
Absorb the true scale of the universe through escalating numbers and distances
Walk away genuinely knowing things about space they didn't know before - and not minding at all
Key Design Rule
Every real fact must be presented with absurd commentary that makes it MORE memorable, not less. The humor is the delivery mechanism for the education.

2.4 Building Cost Formula
All building costs follow a standard exponential curve used by most idle games. This ensures each purchase feels meaningful without being impossible:
Cost Formula
Cost(n) = BaseCost x 1.15^(n) where n = number of that building already owned. Each building costs approximately 15% more than the last.
Example: If an Observatory has a base cost of 100 ?, the 10th Observatory costs 100 x 1.15^10 = ~405 ?. The 50th costs ~1,083,000 ?.
2.5 Upgrade Purchase Model
Upgrades are one-time purchases that appear in the upgrade panel when certain thresholds are reached. Unlike buildings, they do not scale - they have a fixed cost and a fixed effect. They are permanent and persist until prestige (at which point the player must re-buy them - this is intentional, giving early prestige runs a clear goal).
SECTION 3 - BUILDINGS
3. Buildings
Buildings are the core passive income generators of Nebula Brewery. Each building has a name, a base cost, a base production rate per second, a description, a set of upgrade tiers, and a set of flavor text entries that change as you own more of that building. Buildings appear in the shop in order - the player must buy at least one of the previous building before the next unlocks.
Naming Convention
Each building name follows the theme of cosmic scale - starting local (Observatory watching the sky) and ending incomprehensible (Singularity Tap drawing from black holes). The humor escalates proportionally.

SECTION 5 - SPACECRAFT & EXPLORATION SYSTEM
5. Spacecraft & Exploration System
The Spacecraft and Exploration system is the second major pillar of Nebula Brewery. While buildings handle passive production, spacecraft unlock the universe - literally. Each spacecraft has a range that defines which celestial zones it can explore. Each zone contains real astronomical facts, unique ingredients for production bonuses, and narrative flavor text with absurdist commentary.
Design Intent
The exploration system should feel like actual discovery. When a player unlocks a new zone, there should be a genuine moment of wonder - both from the game's humor and from the real facts presented. The educational content is most effective precisely at this moment of novelty.

SECTION 6 - THE NEWS TICKER
6. The News Ticker
The News Ticker scrolls across the top of the game screen at all times. It displays rotating headlines that combine absurdist humor with real space facts, react to player milestones, and occasionally hint at hidden content. This is one of the most important features for long-term player retention - it makes the world feel alive.
Writing Rule
Every news entry should make the player smirk, occasionally make them look something up, and always make the game world feel bigger than what's on screen. The best ones are headlines that sound real until the last four words.

General Space News - absurdist reframings of real space facts

Patron Reports - news about your cosmic bar's clientele

SECTION 8 - PRESTIGE: THE BIG BANG RESET
8. Prestige - The Big Bang Reset
The Big Bang Reset is Nebula Brewery's prestige mechanic. When a player has earned enough stardust in a single run (threshold: ~500 million in the first run), they unlock the ability to collapse their entire operation and start over - but not from nothing.
8.1 How It Works
Player reaches prestige threshold for the current run
The "Big Bang" button activates - large, dramatic, impossible to miss
Player is shown a confirmation screen detailing exactly what they will lose and keep
On confirmation: all buildings, upgrades, and stardust are reset to zero
Player keeps: Cosmic Memory points (prestige currency), any unlocked secret zones, certain permanent achievements
New run begins ~25% faster than the previous one thanks to Cosmic Memory multipliers
Subsequent prestige thresholds are higher - each run must reach a larger total to prestige again
8.2 Big Bang Reset Screen
"The universe is 13.8 billion years old. It started with a single, incomprehensible explosion of everything from nothing. You're about to do something similar. But smaller. On purpose."
Display on the Big Bang screen:
Total stardust earned this run
Buildings and upgrades that will be lost
Cosmic Memory points that will be earned
Production multiplier for the next run
A real astronomical fact about the Big Bang - different each time
8.3 Big Bang Real Facts (Displayed on Reset Screen)
#
Fact
1
The Big Bang was not an explosion in space - it was an expansion of space itself. There was no center. Every point was the origin.
2
In the first 3 minutes after the Big Bang, the universe formed all the hydrogen and helium that exists. Everything else was made later, inside stars.
3
The cosmic microwave background radiation - the afterglow of the Big Bang - still fills the entire universe today. Your microwave picks up a small fraction of it as static.
4
The Big Bang did not produce heavy elements. Iron, gold, uranium - all of it was forged in the cores of dying stars and scattered by supernovae. You are made of those elements.
5
Before the Big Bang, there was no "before." Time itself began with the Big Bang. Asking "what was there before" is like asking "what is south of the South Pole."
6
The observable universe contains approximately 2 trillion galaxies. Each was seeded by tiny quantum fluctuations in the first fraction of a second after the Big Bang.
7
In the first second after the Big Bang, the temperature was 10 billion degrees Celsius. In the first trillionth of a second, it may have been infinite. We don't have equations for infinite.

To keep prestige runs from feeling identical, each new run introduces at least one new element:
New news ticker entries unlock (late-game and prestige-specific headlines)
Patron dialogue shifts - regulars comment on the reset ("You're back. You look different. Same eyes.")
One new secret achievement becomes discoverable per prestige
After 5 prestiges, a new building tier becomes available (Tier 11, currently redacted)

CLASSIFIED
Void Essence, when collected from Zone ∞, does not increase production. Instead, it triggers a visual event: the game screen slowly fills with stars until it looks like a real photograph of space. All UI fades. The ticker displays one final message: "You found it. The thing between things. The space between spaces. It was always here. You just needed to look." After 30 seconds, the game resumes normally. There is no production bonus. This is intentional. Some things are not about the numbers.

The Big Bang Reset button should glow and throb with increasing intensity as the player gets closer to the prestige threshold

SECTION 12 - TECHNICAL ARCHITECTURE
12. Technical Architecture
Nebula Brewery v1 is a single-file browser game. No server, no database, no login required. Progressive enhancement: works without an account, optionally syncs via localStorage.

12.2 Save System
All game state is saved to localStorage every 30 seconds and on tab close/blur
Save data is a single JSON object: { stardust, buildings, upgrades, spacecraft, zones, achievements, prestige, settings, timestamp }
Players can export their save as a base64 string for backup/sharing
Players can import a save string to restore progress
On prestige, save is overwritten with fresh state + preserved prestige data
